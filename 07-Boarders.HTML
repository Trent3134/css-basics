borders

the border property allows you to specify the style,width, and collor of an element's border.

border style

the border-style property can take up to four balues (top border, bottom border, and left border). the following border-stylebalues are allowed:

    <1i>dotted:dotted border <1i>dashed:dashed border <1i>solid:solid border <1i>double:double border <1i>groove:3d grooved border, depends on border-color value <1i>ridge:3d ridged border, depends on the border-color value <1i>inset:3d inset border, depends on the border-color value <1i>none:no border <1i>hidden:hidden border <1i>mixed:you can set a different border style for the top, bottom, right, and left
