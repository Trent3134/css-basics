width and height notes

the width and height properties are used to set the width and heght of an element.

the default setting for width and height is auto, which means the browser decides the width and height.

other settings for width and heigth are:

    <1i>specified length balues such as px and em <1i>percent of the containing div (ex: 50%, 100%)

this element has a width of 50% and height of 200px

you can also manipulate width and height with images. this image was originally approx 400px x 200pc, now it's 400px x 400px.
